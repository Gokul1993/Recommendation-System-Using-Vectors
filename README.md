# 🚀 Recommendation System using Vectors (Python)

## 📌 Overview

This project is a **simple product recommendation system** built using
Python.\
It uses **vector representation of users and products** and applies
**Cosine Similarity** to recommend the most relevant products to users.

Inspired by real-world systems like **Amazon, Netflix, and YouTube
recommendations**.

------------------------------------------------------------------------

## ⚙️ Tech Stack

-   🐍 Python\
-   🐼 Pandas\
-   🔢 NumPy\
-   📊 Cosine Similarity\
-   📁 CSV Dataset (User & Product vectors)

------------------------------------------------------------------------

## 🧠 Core Concept

Each user and product is converted into a **vector in multi-dimensional
space**.

Cosine Similarity formula:

cos(θ) = (A·B) / (\|A\|\|B\|)

-   Higher cosine value → More similar → Better recommendation\
-   Lower cosine value → Less relevant product
